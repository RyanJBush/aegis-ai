Implemented Scan Categories

Injection Patterns

Regex checks for SQLi and XSS payload signatures in input text.

Auth & Access Signals

Rules for weak auth settings, basic auth usage, and permissive access flags.

Config & Header Hardening

Checks for insecure config defaults and missing/permissive HTTP security headers.

Secret Exposure Patterns

Detection of token/private-key style strings and sensitive credential-like content.
